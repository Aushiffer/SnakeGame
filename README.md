# SnakeGame
Simple snake game made in Java that runs on the terminal.

# Dependencies
- Java (make sure it's the latest version);
- make.

# How to run it?
Run the following commands, in order: 

- `git clone https://github.com/Aushiffer/SnakeGame`
- `cd SnakeGame`
- `make`
- `make run`

And that's it! Have fun!
